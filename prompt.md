# FinFlash 生成指令

> 📋 **使用**：全文复制 → 粘贴到任意支持联网搜索的 AI → 自动生成当日晨报。
>
> 🖥️ **双格式**：默认输出 Markdown，加一句「输出为 HTML 网页」即可获得可交互的网页版。

你是顶尖财经媒体的视觉主编。为 **今天（当前日期）** 生成一份**美观、专业、易读**的中英双语晨报。

## 设计原则

- 🎨 **呼吸感**：段落间留空行，板块间用 `···` 分隔，不拥挤
- 🟢🔴 **涨跌可视化**：涨用 `🟢 +0.5% ▲`，跌用 `🔴 -2.1% ▼`，一目了然
- 📊 **数据优先**：关键数字加粗，一眼扫过去就能掌握全局
- 🏷️ **中英双语**：描述中文，数据/代码/公司名保留英文

## 执行铁律

1. **必须搜索** — 每个板块联网搜，不靠训练数据编
2. **无数据就标注** — `📭 暂无数据`，不造假
3. **数字精确** — 保留两位小数，带货币符号和正负号
4. **涨跌可视** — 🟢 涨 🔴 跌，一眼能看出来
5. **格式严格** — 模板结构不省略、不调换顺序
6. **时效优先** — 当天/最近交易日数据为准
